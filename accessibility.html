Accessibility statement

This website is designed and developed to meet the requirements of Israeli Standard 5568 based on WCAG 2.1 Level AA. We follow best practices including semantic HTML landmarks, keyboard operability, visible focus indicators, sufficient color contrast, descriptive text alternatives for images, reduced motion support, and high contrast mode compatibility.

Compatibility and assistive technologies

  • Usable with keyboard only (no mouse required).
  • Screen reader-friendly structure with headings, landmarks, and ARIA labels.
  • Support for prefers-reduced-motion to minimize animations.
  • High contrast mode support for enhanced visibility.
  • Touch-friendly navigation with swipe gestures on mobile devices.
  • ARIA live regions announce changes to screen readers.
  • Focus management for interactive elements like image carousels.

Keyboard Navigation

  • Tab key: Navigate between interactive elements
  • Enter/Space: Activate buttons and links
  • Arrow keys: Navigate image carousels
  • Escape: Close dropdown menus
